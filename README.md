# taskbot
A Discord taskbot that uses GitHub tasks as a backing store.

## config

**`.env`** - Contains secrets loaded from an ENV file. Requires `DISCORD_TOKEN` set with a valid Discord app token:

    DISCORD_TOKEN=SomE-ToK-EN
